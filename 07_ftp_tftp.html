FTP & TFTP

1. FTP サーバー（vsftpd）

変数を設定

bash
FTP_USER="admin"
FTP_PASS="password"

パッケージのインストール & ディレクトリ作成

bash
dnf -y install vsftpd
mkdir -p ftpd && chmod 777 ftpd
useradd $FTP_USER
echo "$FTP_PASS" | passwd --stdin $FTP_USER

vsftpd.conf の設定

bash
cp -p /etc/vsftpd/vsftpd.conf /etc/vsftpd/vsftpd.conf.org

cat << 'EOF' > /etc/vsftpd/vsftpd.conf
# Basic
anonymous_enable=NO
local_enable=YES
write_enable=YES
local_umask=022
dirmessage_enable=YES
xferlog_enable=YES
connect_from_port_20=YES
xferlog_std_format=YES

# Connection
chroot_local_user=YES
allow_writeable_chroot=YES

# Server
listen=YES
listen_ipv6=NO
pam_service_name=vsftpd
userlist_enable=YES
tcp_wrappers=YES

# mode: passive
pasv_enable=YES
pasv_min_port=30000
pasv_max_port=30010
EOF

サービス起動 & 確認

bash
systemctl enable --now vsftpd
lsof -i:21 && systemctl status vsftpd --no-pager

2. TFTP サーバー（tftp-server）

パッケージのインストール

bash
dnf -y install tftp-server tftp
ln -s /var/lib/tftpboot ./tftpd && chmod 777 ./tftpd

サービス起動 & 確認

bash
systemctl enable --now tftp.socket
lsof -i:69 && systemctl status tftp.socket --no-pager

3. テスト

Juniper

junos
# FTP
file copy ftp://$FTP_USER:$FTP_PASS@$MY_SERVER_IP/filename /var/tmp/filename
# TFTP
file copy tftp://$MY_SERVER_IP/filename /var/tmp/filename

Apresia / Alaxala

switch cli
# FTP
copy ftp://$FTP_USER:$FTP_PASS@$MY_SERVER_IP/filename flash
# TFTP
copy tftp://$MY_SERVER_IP/filename flash
