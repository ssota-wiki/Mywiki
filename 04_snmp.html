ℹ 前提 — rsyslog が設定済みであること。SNMP Trap は rsyslog の local6 ファシリティ経由でファイル出力します。03. SYSLOG を先に設定してください。

1. SNMP-Get クライアント（snmpd）

パッケージのインストール

bash
dnf -y update && dnf -y install net-snmp net-snmp-utils

変数を設定

bash
SNMP_INFO='SP-LAB / Local'
SNMP_COMMUNITY='public'
SNMP_TARGET_IP='localhost'
SNMP_SOURCE_IP='91.1.128.10'
SNMP_SUBNET_1='90.1.128.0/24'

バックアップ

bash
cp -p /etc/snmp/snmpd.conf /etc/snmp/snmpd.conf.org

snmpd.conf の設定

bash
cat << EOF > /etc/snmp/snmpd.conf
# system info
syslocation ${SNMP_INFO}

# com2sec | sec.name    | source           | community
com2sec     sec_private   127.0.0.1          public
com2sec     sec_private   ${SNMP_SUBNET_1}   ${SNMP_COMMUNITY}

# group
group     grp_private   v1              sec_private
group     grp_private   v2c             sec_private

# view
view all      included    .1                                 80

# access
access grp_private   ""        any         noauth      exact    all    all     all

interface eth0 6 1000000000
disk /
disk /usr
disk /var
EOF

サービス起動 & テスト

bash
systemctl enable --now snmpd.service
lsof -i:161

# 動作テスト
snmpwalk -d -v 2c -c ${SNMP_COMMUNITY} ${SNMP_TARGET_IP} sysName

# 送信元IPを指定してテスト
snmpwalk -d -v 2c -c ${SNMP_COMMUNITY} ${SNMP_TARGET_IP} --clientaddr=${SNMP_SOURCE_IP} sysName

2. SNMP-Trap サーバー（snmptrapd）

変数を設定

bash
SNMP_COMMUNITY="public"
SNMP_FACILITY="6"    # local6 → rsyslog で /var/log/snmptrap/ へ出力

バックアップ

bash
cp -p /etc/snmp/snmptrapd.conf /etc/snmp/snmptrapd.conf.org

起動オプション設定

bash
cat << EOF > /etc/sysconfig/snmptrapd
OPTIONS="-Ls${SNMP_FACILITY}d -p /var/run/snmptrapd.pid"
EOF

snmptrapd.conf の設定

bash
cat << EOF > /etc/snmp/snmptrapd.conf
authCommunity log,execute,net ${SNMP_COMMUNITY}
EOF

サービス起動 & テスト

bash
systemctl enable --now snmptrapd.service
lsof -i:162

# 動作テスト（rsyslog 経由で snmptrap ログに出力されるか確認）
logger -p local${SNMP_FACILITY}.notice -t rsyslogd "TEST-03: Testing SNMP Trap"
grep -E "Testing" /var/log/snmptrap/*.log
